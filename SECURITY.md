# Security Policy

## Supported versions

Security fixes are provided for actively maintained public repositories in the [ioai-tech](https://github.com/ioai-tech) organization. Check each repository's README or release notes for supported versions.

## Reporting a vulnerability

**Please do not open public GitHub issues for security vulnerabilities.**

Report security issues to **[io@io-ai.tech](mailto:io@io-ai.tech)** with:

- A description of the issue and potential impact
- Steps to reproduce
- Affected repository and version or commit, if known

We aim to acknowledge reports within **5 business days** and will coordinate disclosure timelines with you.

## Safe harbor

We support responsible disclosure and will not pursue legal action against researchers who follow this policy in good faith.
