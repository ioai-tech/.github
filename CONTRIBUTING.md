# Contributing to IO-AI.tech Open Source

Thank you for your interest in IO-AI.tech projects. We welcome bug reports, documentation improvements, and pull requests.

## Before you start

1. Search [existing issues](https://github.com/ioai-tech) to avoid duplicates.
2. For large changes, open an issue first to discuss scope.
3. Each repository may have its own `CONTRIBUTING.md` — follow the repo-specific guide when present.

**Example:** [rosview/CONTRIBUTING.md](https://github.com/ioai-tech/rosview/blob/main/CONTRIBUTING.md)

## Pull request checklist

- [ ] Changes are focused on a single concern
- [ ] Commit messages explain *why*, not only *what*
- [ ] Documentation is updated when behavior changes
- [ ] Tests pass locally when the repo has a test suite

## Code of conduct

Be respectful and constructive. We prioritize clarity, craftsmanship, and outcomes.
